### 1. GitHub = 주방 (레시피 보관소)

선생님이 `.qmd` 파일(강의 레시피)을 여기 올려요. 코드만 있고 아직 요리(웹사이트)는 아니에요.

---

### 2. GitHub Actions = 주방 직원

선생님이 push(재료 납품)하면 자동으로 깨어나서:
```
R 설치 → 패키지 설치 → quarto render → HTML 파일 생성
```
이걸 자동으로 해줘요. 선생님은 손 안 대도 됨.

---

### 3. GitHub Pages = 홀 (손님이 보는 곳)

주방 직원이 만든 HTML 파일을 인터넷에 올려놓는 곳이에요.  
`https://kkonoo.github.io/2_linear_algebra/` 가 바로 이 주소예요.

---

### 4. Cloudflare Worker = 비밀 창고 지킴이

학생이 AI한테 질문하면:
```
학생 브라우저 → Cloudflare Worker → Anthropic API
```
API 키가 Cloudflare 안에 숨겨져 있어서 학생은 절대 못 봐요.  
선생님 카드로 결제되는 식당 전용 재료 창고인데, 문 앞에 지킴이가 서있는 거예요.

---

### 5. Deploy = 오픈

주방에서 요리가 완성되면 홀에 내보내는 것. "이제 손님 받아요" 하는 순간이에요.

---

## 전체 흐름 한 줄로

```
선생님이 .qmd 수정 → push → Actions가 자동 빌드 → 
Pages에 올라감 → 학생 접속 → AI 질문 → 
Worker 통해서 Claude 답변
```

push 한 번만 하면 나머지는 다 자동이에요!


## github workflow
push 감지
  ↓
ubuntu 컴퓨터 빌려옴 (GitHub 서버)
